## âš ï¸ Common Mistakes

1. **Forgetting lists start at 0** - `fruits[1]` is the second item, not the first!
2. **Using append() for multiple items** - Use `extend()` instead
3. **Accessing non-existent dictionary keys** - Use `get()` to avoid crashes
4. **Modifying a list while looping** - Create a copy first
5. **Confusing sort() and sorted()** - `sort()` changes original, `sorted()` creates new list

## ğŸ’¡ Key Concepts

- **Lists**: Ordered collections of items (like a shopping list)
- **Dictionaries**: Key-value pairs (like a contact list with names and info)
- **Indexing**: Lists use numbers (0, 1, 2...), dictionaries use keys
- **Slicing**: Get parts of a list with `[start:end]`
- **Negative indexing**: Count from the end with `-1, -2, -3...`
