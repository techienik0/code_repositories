# Episode 1: Why Java?

Introduction episode - why Java is used everywhere in enterprise and backend development.

## What We Covered

- Why companies like Netflix, Amazon use Java
- How Java compares to other languages
- What we'll build in this series

## Code Example

Just showed a simple Hello World to demonstrate Java syntax:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

## Key Points

- Java runs on any platform (write once, run anywhere)
- Built for large-scale applications
- Huge ecosystem (Spring Boot, Hibernate, etc.)
- Industry standard for backend development
